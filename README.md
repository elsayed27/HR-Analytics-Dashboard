# 📊 Sales Performance Analytics Dashboard | Power BI

## 🚀 Overview

This project presents a professional **Power BI Dashboard** built to transform raw sales data into clear, actionable business insights.
The dashboard enables users to monitor company performance, track KPIs, evaluate profitability, and uncover growth opportunities through interactive visual storytelling.

Designed for business managers, analysts, and decision-makers who need fast and accurate performance analysis.

---

## 🎯 Business Goals

* Measure overall sales performance
* Track profit growth and margins
* Discover top-selling products
* Compare regional performance
* Analyze customer demand trends
* Improve strategic decision-making
* Identify underperforming areas

---

## 🛠 Tools & Skills Applied

* Power BI Desktop
* Power Query (Data Cleaning & Transformation)
* DAX (Calculated Measures & KPIs)
* Data Modeling
* Interactive Dashboard Design
* Business Intelligence Reporting

# 📂 Dataset

The dataset contains historical sales transaction records used to evaluate business performance across multiple dimensions.

### 📊 Key Columns

* **Order Date** → Transaction date
* **Product Name** → Sold product
* **Category** → Product classification
* **Region** → Geographic sales area
* **Sales** → Revenue generated
* **Profit** → Net earnings
* **Quantity** → Units sold

---

## 📌 Data Value

This dataset allows deep analysis of:
